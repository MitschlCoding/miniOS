## Dependencies

-   qemu-system
-   nasm
-   build-essential
-   libtool
-   gcc-multilib
-   libc6-dev-i386

## Commands

If you have make installed and use Ubuntu, you can use the following command to install the dependencies:

```bash
make installDeps
```

To build and run the OS using qemu use:

```bash
make run
```

To clean all object-files and bins use:

```bash
make clean
```
